# 전개/배포

- 소프트웨어 전개(software deployment)는 **소프트웨어 시스템을 사용할 수 있도록 만드는 모든 활동**을 말한다.
- 소프트웨어 배포(software distribution)는 **최종 사용자에게 소프트웨어를 전달하는 과정**이다.



### 무엇을

- 서버컴퓨터에서 요청과 응답을 처리할 프로그램 (소프트 웨어)



### 언제

while alive:

1. 분석, 계획, 설계 -> 개발
2. 개발	->개발
3. 테스트 -> 개발
4. 배포(배치) -> 운영
5. 운영 -> 운영



### 누가, 어디에

- Native App
  - 사용자가 사용자 컴퓨터에
- Web App
  - 제공자가 제공자 컴퓨터에



- 컴퓨터 빌려주는 플랫폼
  - AWS, Azure, GCP
